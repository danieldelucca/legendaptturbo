# EXPLICAÇÕES E COMO USAR
 
# Exemplo 1:

# Se há uma legenda com a frase:		Eu vou pensar nisso.

# E há uma linha neste arquivo com esses termos abaixo:
# Vou pensar nisso. = Vou pensar.

# O item listado acima não substituirá essa frase, pois 'Vou' tem V maiúsculo, ao contrário de 'vou' na frase de exemplo


# Exemplo 2:

# Se há uma legenda com a frase:		Eu vou pensar nisso.

# Se há uma linha neste arquivo com esses termos abaixo:
# Eu vou pensar nisso = Vou pensar.


# O item listado acima não substituirá essa frase, pois ela termina com '.' e o termo listado, não.
# Sem pontuação no final do termo, ele irá substituir o mesmo termo seguido de espaço ou em final de linha, sem pontuação
# Neste caso, para que o termo com ponto também seja substituído, adicione uma nova linha com '.'

# Essa lógica nos traz mais controle sobre as substituições - se queremos substituir apenas o começo de uma frase, uma frase inteira, etc
# Para listar termos a serem substituídos independente de maiúsculas/minúsculas, use o arquivo substituições_case_insensitive.txt

# Para adicionar novos termos, separe-os por =
# Se precisar usar espaço no começo ou final, coloque o termo entre ' '
# Para desativar alguma linha, basta colocar # no começo da linha ou deletá-la
# Para colocar explicações, coloque # na linha, depois das palavras listadas
